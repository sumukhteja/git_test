Things we can try within lists:
  1. Start attribute

    Syntax: ol start="30" - list starts with numbering from 30

  2. Reversed attribute

    Syntax: ol reversed - list starts with reversed numbering

  3. Value attribute

    Syntax: li value="9" - when used on the 3rd list element, the order changes to 9 and the next list element number is 10 and so on

New type of list: Description list

This uses d1 elements to open and close. Inside the d1 element, dt and dd elements are used.

dt
dt means definition term
dd
dd means description
